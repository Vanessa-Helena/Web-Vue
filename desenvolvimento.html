Desenvolvimento

Basicamente, o Vue.Js funciona como um auxiliar no desenvolvimento do Front-End, focando em trabalhar juntamente ao HTML. Se o código HTML for alterado, ele automaticamente altera no Vue, deixando assim a integração do framework com o HTML mais veloz e precisa!

Através do Vue conseguimos alterar imediatamente qualquer aspecto do nosso HTML em tempo real, somente precisamos de algumas pequenas linhas de código para verificar o funcionamento do Vue integrado ao HTML.

Exemplos Práticos

    div id="app"
      {{ message }}
    div

    script
      var app = new Vue({
        el: '#app',
        data: {
          message: 'Hello Vue!'
        }
      })
    script
