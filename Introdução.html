Introdução

O Vue Js é um framework JavaScript para criação de interface de usuários e single page aplications. Teve sua origem nos bastidores do Google (mesmo palco em que foi projetado outro framework que se chama Angular). Seu criador foi Evan You, que ao trabalhar com o Angular resolveu desenvolver sua própria biblioteca do framework somente com as coisas que ele achava que eram válidas e boas do angular, e foi assim que se deu início a criação do Vue Js até dia 07 Junho de 2013.

Durante seu desenvolvimento, Evan pegou inspiração em diversos outros frameworks e tecnologias da web moderna como por exemplo os Web components, as diretivas, templates data bind do angular entre outras ferramentas associadas a essa tecnologia como por exemplo o single file component que é um dos maiores pontos a se destacar no Vue Js que tem como inspiração outro projeto nomeado de RACTIVE

Tvan, após trabalhar em diversos projetos utilizando o framework angular, resolveu então extrair os pontos que mais agradava a ele e desenvolver um framework específico para suas necessidades e também com uma maior leveza nasce assim o Vue Js em fevereiro de 2014. Em 2015 um framework PHP chamado LARAVEL incorporou o Vue JS em seu projeto, resultando assim numa grande comunidade de adeptos ao novo framework JavaScript. Com o aumento dos usuários do framework e uma comunidade sólida, Evan decidiu se desligar do Google trabalhando integralmente para o Vue Js tornando-o um dos maiores frameworks utilizados em todo o mundo alcançando as melhores posições do Github!!
